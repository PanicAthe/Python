## 2. 핵심 개념 정리

### 2-1. 로지스틱 회귀(Logistic Regression)
- 분류 문제에서 클래스 확률을 예측하는 선형 모델
- 내부적으로는 입력의 선형결합 z를 만든 뒤, 이진 분류에서는 시그모이드 함수를 통과시켜 확률로 변환함
- 다중 분류에서는 클래스별 점수를 소프트맥스 형태 확률로 해석함

### 2-2. 시그모이드 함수
- 식: p = 1 / (1 + e^(-z))
- z가 커질수록 p는 1에 가까워지고, z가 작아질수록 p는 0에 가까워짐
- 분류 경계 근처(z=0, p=0.5)에서 예측이 가장 민감하게 변함

### 2-3. Binary Cross-Entropy(로그 손실)
- 식: -(y log(p) + (1-y) log(1-p))
- 정답이 1인데 p가 작거나, 정답이 0인데 p가 크면 손실이 급격히 커짐
- p=0 또는 p=1 근처에서 log 발산이 있어 코드에서는 보통 작은 epsilon을 두고 계산함
- 이번 노트에서는 y=1, y=0, y=0.5(이진 엔트로피) 곡선을 함께 그려 손실 형태를 비교함

## 5. 공부 포인트
- 로지스틱 회귀는 "선형결합 + 확률변환" 구조라 해석이 비교적 쉬움
- 스케일링을 하지 않으면 특성 단위 차이 때문에 학습 안정성과 성능이 흔들릴 수 있음
- 다중 분류에서 predict뿐 아니라 predict_proba를 함께 보는 습관이 중요함
- 손실함수 그래프를 직접 그리면 BCE가 왜 확률 오차에 민감한지 직관적으로 이해됨
