# 2_선형회귀 개념/공부 요약

## 1. 이 노트북에서 한 일 (큰 흐름)
- 회귀분석이 무엇인지 개념을 설명함
- length와 weight 데이터를 numpy 배열로 직접 만듦
- 산점도로 두 변수의 관계를 시각화함
- train_test_split으로 학습/테스트 데이터를 분리함
- 선형회귀 모델에 맞게 입력을 2차원 배열로 reshape함
- LinearRegression으로 모델을 학습함
- 계수(coef_)와 절편(intercept_)을 확인함
- 새로운 값 31, 45, 12에 대한 예측을 수행함
- 직선 그래프를 그려 데이터와 회귀식을 함께 확인함
- Cost/Loss function, 손실함수 개념을 이어서 학습함

## 2. 핵심 개념 정리

### 2-1. 회귀분석(Regression)
- 연속형 값을 예측하는 지도학습 문제
- 분류처럼 클래스가 아니라 숫자 자체를 예측함
- 이 노트북에서는 길이(length)로 무게(weight)를 예측하는 문제를 다룸

### 2-2. 선형회귀(Linear Regression)
- 입력 x와 출력 y 사이의 관계를 직선으로 근사함
- 기본 형태는 y = ax + b
- a는 기울기, b는 절편
- 데이터에 가장 잘 맞는 직선을 찾는 것이 목표

### 2-3. 손실함수 / Cost Function
- 모델이 예측한 값과 실제 값의 차이를 수치로 표현한 것
- 선형회귀는 보통 오차가 작은 직선을 찾기 위해 손실을 최소화함
- 실제 값과 예측 값 차이가 클수록 손실이 커짐

## 3. 코드 흐름 요약

### 3-1. 데이터 만들기
- length: 물체의 길이 데이터
- weight: 해당 길이에 대응하는 무게 데이터
- 산점도로 보면 길이가 커질수록 무게도 커지는 경향이 보임

### 3-2. 데이터 분할
- train_test_split(length, weight, random_state=11)
- 학습용과 테스트용으로 나눠서 일반화 성능을 확인함
- reshape(-1, 1)로 1차원 배열을 2차원 입력 형식으로 바꿈

### 3-3. 모델 학습
- LinearRegression() 생성
- fit(x_train, y_train)으로 학습
- coef_와 intercept_로 회귀식 확인

### 3-4. 예측과 시각화
- new = np.array([31, 45, 12]).reshape(-1, 1)
- predict로 새 데이터의 무게를 예측함
- scatter와 plot을 함께 그려 회귀선을 확인함

## 4. 실행 결과 핵심
- 학습된 기울기(coef_): 36.14738611889727
- 학습된 절편(intercept_): -631.5549218539231
- 예측값
  - 31 -> 489.01404783
  - 45 -> 995.0774535
  - 12 -> -197.78628843

해석
- 길이가 길어질수록 무게가 커지는 양의 선형 관계를 학습함
- 12처럼 범위 밖에 가까운 값은 예측 결과가 음수가 나올 수 있는데, 이런 점도 선형회귀의 한계로 볼 수 있음

## 5. 공부 포인트
- 선형회귀는 입력과 출력이 거의 직선 관계일 때 해석이 쉬움
- 데이터가 직선형이 아니면 성능이 제한될 수 있음
- reshape를 빼먹으면 sklearn 모델이 기대하는 입력 형태와 맞지 않음
- train/test 분리를 해야 모델이 외운 값이 아니라 새로운 데이터에 얼마나 잘 맞는지 볼 수 있음
