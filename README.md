## 개념 구분

| 구분 | 예시 | 역할 |
|---|---|---|
| 전처리 | StandardScaler | 입력 특성의 단위와 범위를 맞춰서 학습이 한쪽 값에 끌리지 않게 만드는 준비 단계입니다. 특히 거리 기반 모델(KNN)이나 경사하강법을 쓰는 모델에서 중요합니다. |
| 모델 | KNN, 선형회귀, 로지스틱 회귀, 신경망 | 입력을 받아 예측값을 만드는 본체입니다. KNN은 가까운 이웃을, 선형회귀는 직선식을, 로지스틱 회귀는 확률을, 신경망은 층을 쌓은 비선형 변환을 사용합니다. |
| 활성화 함수 | Sigmoid, ReLU, Softmax | 신경망 내부나 출력층에서 값의 형태를 바꾸는 함수입니다. 비선형성을 넣거나, 0~1 확률처럼 해석되게 하거나, 여러 클래스 점수를 확률 분포로 바꿉니다. |
| 손실함수 | MSE, Binary Cross-Entropy | 모델 예측이 정답과 얼마나 다른지 수치로 계산하는 벌점입니다. 손실이 작아질수록 예측이 좋아지고, 옵티마이저가 이 값을 줄이도록 학습합니다. |
| 옵티마이저 | SGD, Adam | 손실함수를 줄이기 위해 가중치와 편향을 실제로 업데이트하는 규칙입니다. 모델이 아니라 학습 방법이며, gradient를 보고 파라미터를 조금씩 수정합니다. |
| 문제 예시 | XOR, 와인 분류 | 모델이나 함수가 아니라 풀어야 할 학습 과제입니다. XOR는 선형 분리가 안 되는 대표 문제이고, 와인 분류는 실제 다중 분류 실습 예시입니다. |

## 빠른 참고

- KNN은 거리 계산이 핵심이라 전처리 영향이 큽니다.
- 선형회귀와 로지스틱 회귀는 둘 다 선형결합을 쓰지만, 하나는 연속값 예측이고 다른 하나는 분류 확률 예측입니다.
- XOR는 모델이 아니라 신경망이 왜 필요한지 보여주는 비선형 문제 예시입니다.
- Softmax와 Sigmoid는 둘 다 출력 변환 함수지만, Softmax는 다중 분류, Sigmoid는 이진 분류에 더 직접적으로 대응합니다.
